# cn-site-notes

## 项目简介

`cn-site-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面本身为静态 HTML 文件，涵盖各类信息记录与内容展示，不针对任何特定网站或域名。仓库主要作为页面存档与索引的载体，方便浏览与检索。

## 目录说明

```
cn-site-notes/
├── pages/           # 存放所有独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...  
├── assets/          # 页面引用的外部资源（样式、图片、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
├── index.html       # 仓库首页（可选，用于列出页面索引）
└── README.md        # 本文件
```

- **pages/**：所有归档的 HTML 页面，每个文件独立存在，命名清晰。
- **assets/**：页面所需的公共资源，按类型分目录存放。
- **index.html**：可选，用于提供页面列表或导航。

## 页面归档说明

- 每个页面均为独立自包含的 HTML 文件，可单独打开浏览。
- 页面内容已做脱敏与去标识化处理，不包含具体域名或可定位到特定网站的信息。
- 归档仅保留页面结构与展示内容，不涉及任何动态交互或后台服务。

## 维护说明

- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保引用的资源路径正确。
- 资源文件请优先放入 `assets/` 对应子目录，避免分散存放。
- 不建议修改已有页面的文件名或路径，以免影响历史引用。
- 本仓库不负责页面内容的时效性审核，仅作为存档与展示用途。

## 许可

本仓库仅用于内容归档与展示，不构成对页面内容的任何声明或授权。如有版权或其他问题，请通过仓库 Issues 联系。
